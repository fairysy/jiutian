
# 玖天渗透测试工具箱

玖天渗透测试工具箱 是一个面向安卓平台的渗透测试工具合集，适用于合法授权的内网安全演练、CTF 比赛环境下的信息收集、漏洞利用与数据分析等场景。

## ✨ 功能特色

- 🔍 局域网设备扫描（支持 ARP / Ping / SNMP / UPnP）
- 🌐 端口扫描、Banner 抓取、常见服务识别
- 🧰 内置常见漏洞检测工具（如弱口令、Web 指纹识别等）
- 🧾 自动化 Payload 构造与 APK 生成
- 🧱 支持插件式扩展开发
